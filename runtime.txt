502 FIX STABIL SERVER

Bu paket 502 hatasını azaltmak için:
- Login sayfasında database bekletmez.
- Database hatasını panel içinde gösterir.
- Render yanlış yapılandırılsa bile python app.py ile açılır.
- Gunicorn ayarı daha uzun timeout kullanır.

Render Start Command:
gunicorn --workers 1 --threads 2 --timeout 180 app:app

Build Command:
pip install -r requirements.txt

Environment:
DATABASE_URL = Neon connection string

Deploy:
Manual Deploy > Clear build cache & deploy

Test:
/api/health
